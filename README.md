### 编译静态网站

您可以通过以下步骤构建本项目，生成静态网页素材：

0.前置条件(低于所示的版本暂未测试。如有问题恕不维护)：

```text
node >= v22.18.0
npm >= 11.5.2
```

1.安装依赖项

```bash
npm install
```

2.编译

```bash
npm run compile
```

3.发布

生成的静态网站存放于`./public/`下，您可以将其上传至Pages服务或者其他位置供查看
